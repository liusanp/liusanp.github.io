UE5在Windows打包Linux项目
2022-12-05 20:40 folder 资料   label UE5   打包   Linux  

1、在官网下载交叉编译工具链：https://docs.unrealengine.com/5.0/zh-CN/linux-development-requirements-for-unreal-engine/

2、默认安装后在 cmd 中输入 %LINUX_MULTIARCH_ROOT%x86_64-unknown-linux-gnu\bin\clang++ -v 测试是否安装成功

3、重启ue5

4、刷新平台状态

5、如果打包平台前出现Linux图标，则已经可以打包Linux项目

6、选择对应平台打包
